# AI-POWERED-DSA-LEARNING-PLATFORM
An AI-powered DSA learning platform built using Flask and LLMs that provides intelligent doubt solving, adaptive practice, PYQs, AI-based answer evaluation, and personalized learning.
## Features

- 🤖 AI Chatbot using LLM + RAG
- 📚 Topic-wise Learning
- 🎯 Adaptive Practice Mode
- 📝 Previous Year Questions
- ✅ AI-based Answer Evaluation
- 📈 Performance Analytics
- 💡 Personalized Suggestions

## Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript
- JSON
- Groq API
- Llama 3
- RAG
